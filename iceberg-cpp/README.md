# Apache Iceberg C++ Example

A minimal C++ example for the `iceberg-examples` workspace.

## What this example shows

- Defines an Iceberg-style table schema representation in native C++
- Prints schema field IDs, names, types, and required/optional attributes
- Documents current interoperability guidance for C++ users

## Build

```bash
bazel build //iceberg-cpp:iceberg_cpp_example
```

## Run

```bash
bazel run //iceberg-cpp:iceberg_cpp_example
```

## Why this is minimal

Apache Iceberg currently does not publish an official standalone C++ API in the
same style as the Java or Rust APIs. In practice, C++ applications typically
interoperate with Iceberg tables through file formats and metadata generated by
Iceberg-enabled engines (for example, Spark, Flink, Java services, Rust, or
Python).
